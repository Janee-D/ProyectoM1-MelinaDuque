# ProyectoM1-MelinaDuque
🎨 Proyecto Colorfly Studio - M1

Bienvenidos al Generador de Paletas ColorFly Studio!
Este proyecto fue desarrollado como trabajo para el Módulo 1 de Henry.
Consiste en una aplicación web interactiva que genera paletas de colores aleatorias con una interfaz limpia y moderna orientada a branding y diseño.

Podés utilizarla seleccionando la cantidad de colores, el formato de salida (HEX o HSL) mediante el selector y copiar cualquier código al portapapeles haciendo clic sobre el color.

🚀 Tecnologías Utilizadas

HTML — Estructuración semántica del esqueleto de la aplicación, uso de secciones, <header>, <main>, <footer> y atributos de accesibilidad.
CSS — Estilos avanzados con variables nativas (:root), animaciones de transformación (transform, transition), efectos de sombra (box-shadow), diseño responsivo con flexbox y media queries para móviles.
JavaScript — Lógica del negocio completa: algoritmo de generación de colores HEX aleatorios, conversión matemática HEX → HSL, manipulación dinámica del DOM y uso de la API del portapapeles (navigator.clipboard).


🛠️ Características Principales

🌈 Interfaz limpia y moderna — Diseño minimalista con tipografía distintiva y paleta de colores suaves, optimizado para lectura y usabilidad.
🧪 Selector adaptativo — Permite elegir dinámicamente entre 6, 8 o 9 colores simultáneos a renderizar.
⚡ Generación aleatoria — Algoritmo matemático eficiente que asegura la creación de colores HEX válidos en cada interacción.
🔄 Cambio de formato en vivo — Al cambiar entre HEX y HSL, la paleta se regenera automáticamente en el nuevo formato.
📋 Copiado inmediato (UX) — Copiado de códigos de color al portapapeles con un solo clic, asistido por notificaciones emergentes (Toast) con variantes visuales según el tipo de acción.
